# 🌐 Personal Portfolio

A modern, responsive **Personal Portfolio Website** showcasing my skills, projects, education, and experience as a **Full Stack Java Developer**.

## 🚀 Features

* Responsive Design
* Interactive Hero Section
* Typing Animation
* Particle.js Background
* Skills & Experience Timeline
* Project Showcase
* Contact Section
* Social Media Links

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Typed.js
* Particles.js

## 📂 Getting Started

Clone the repository:

```bash
git clone https://github.com/SagarKirtakar/portfolio.git
```

Open `index.html` in your browser to view the portfolio.
